Detailed Description

gpio_drv.h

Introduction

GPIO APIs configures a device pin for use as an INPUT or OUTPUT GPIO. INPUT GPIOs can be configured to generate an interrupt callback on transition.

Usage

The following sequence can be used to configure a GPIO as an output.

Step 1: Call gpiomux_select_gpio(GPIO_PIN_NUM, IO_MUX_ENABLE, OUT_DIR, DRV_CUR_2mA) to configure one GPIO number as Output.

Step 2: Calling gpio_set(GPIO_PIN_NUM, 1) to toggle GPIO low=0 or high=1.

Example

Refer to usr_demo.c under mcusdk/app/demo for more detail example code.
